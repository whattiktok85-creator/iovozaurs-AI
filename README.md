#Guide: How to Make Iovozaurs-AI Operational

1. Install Python: Make sure you have Python 3.12 installed on your PC or Laptop.
2. Run the Script: Start iovozaur_ai.py or double-click starter.bat.
3. Enter API Keys: Follow the console instructions to connect your Telegram Bot Token and Groq Key.
4. Enjoy: Your AI is now online and ready to chat!

GROQ SITE (for create api keys) - console.groq.com/keys
TELEGRAM BOTFATHER (for create bot tokens) - t.me/BotFather
