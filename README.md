#Guide: How to Make Iovozaurs-AI Operational

1. Install Python: Make sure you have Python 3.10+ installed on your PC or Laptop.
2. Install Libraries: Open your terminal and run pip install aiogram groq.
3. Run the Script: Start iovozaur_ai.py or double-click ai.bat.
4. Enter API Keys: Follow the console instructions to connect your Telegram Bot Token and Groq Key.
5. Enjoy: Your AI is now online and ready to chat!

GROQ SITE (for create api keys) - console.groq.com/keys
TELEGRAM BOTFATHER (for create bot tokens) - t.me/BotFather
